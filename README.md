## 📖 简介

智能包裹是一个优化型模组，它不是增加新物品，而是专注于优化原有的背包管理体验。通过智能整理、便捷合成和高效收纳三大核心功能，让你的冒险之旅更加轻松愉快。

## ✨ 主要功能

### 🎯 一键整理背包（核心功能）

在背包或容器界面添加一个**"整理"按钮**，点击即可智能整理所有物品。

**智能整理规则：**
- ✅ **自动堆叠**：自动合并可堆叠的物品
- 📦 **智能分类**：按预设类别分类存放
  - 🔧 工具类（镐、斧、铲等）
  - ⚔️ 武器与盔甲
  - 🍖 食物类
  - 🪵 建材类（方块）
  - ⚙️ 红石类
  - 💎 战利品与贵重物品
  - 🌱 农业类（种子、农作物）
  - 📚 杂物类
- 🎒 **快捷栏优化**：总是将常用工具（镐、剑）放在快捷栏附近
- 🧱 **同类相邻**：将同类方块（石头、深板岩、安山岩等）放在相邻位置

### 🛠️ 便携式合成助手

**快捷键 `G`** 呼出简易合成面板，悬浮在游戏画面一侧。

**核心优势：**
- 🎒 **直接使用背包材料**：无需取出物品
- 🏃 **随时随地合成**：不打开工作台即可合成
- 📋 **常用配方快捷访问**：
  - 基础材料：木棍、木板、火把
  - 工具类：石制/铁制工具
  - 战斗类：箭、弓
  - 建筑类：梯子、栅栏
  - 实用类：箱子、熔炉

**适用场景：**
- 🕳️ 洞穴探险时快速补充工具
- 🏗️ 建筑过程中即时合成材料
- ⚔️ 战斗紧急时刻快速制作武器
- 🏕️ 野外生存时便捷制作必需品

### 🎒 堆叠收纳袋（进阶功能）

合成特殊的**收纳袋**，用于收纳同类别但不同品种的非方块物品。

**收纳袋类型：**

| 收纳袋类型 | 可存放物品 | 特性 |
|-----------|-----------|------|
| 🌾 **种子袋** | 小麦种子、甜菜种子、西瓜种子、南瓜种子等 | 所有种子共享内部槽位 |
| 🎨 **染料袋** | 所有颜色的染料 | 统一收纳，方便取用 |
| 📦 **建材袋** | 各种石头、木头变种 | 每个变种独立槽位，对外只占 1 格 |
| 🍖 **食物袋** | 各类食物食材 | 生食熟食分开存放 |
| ⚗️ **药水袋** | 各类药水、喷溅药水 | 按效果分类存储 |

**使用方式：**
- 👆 **右键点击**：打开收纳袋内部存储空间
- 🎯 **背包整合**：所有内容物收纳后仅占 1 格背包空间
- 🔄 **便捷存取**：支持快速存入和取出操作

## 🎮 使用指南

### 安装要求
- **Minecraft**: 1.21.1
- **Mod Loader**: NeoForge 21.1.220+
- **Java**: Java 21+

### 安装步骤
1. 确保已安装对应版本的 NeoForge
2. 将 `SmartBundle.jar` 文件放入 `.minecraft/mods` 文件夹
3. 启动游戏，享受整理乐趣！

### 操作说明

#### 背包整理
1. 打开背包界面（默认 `E` 键）
2. 点击右下角的 **"整理"按钮**
3. 等待自动整理完成

#### 便携合成
1. 按下 **`G` 键** 打开合成面板
2. 选择需要合成的物品
3. 确认合成（自动消耗背包中的材料）

#### 收纳袋使用
1. 在工作台合成对应的收纳袋
2. 手持收纳袋对物品**右键**即可存入
3. 再次**右键**收纳袋打开内部空间

## 🔧 配置选项

模组提供详细的配置文件（`.minecraft/config/smartbundle.json`），可自定义：

- ⚙️ 整理优先级规则
- ⌨️ 快捷键绑定
- 🎨 UI 界面样式和位置
- 📦 收纳袋容量上限
- 🧩 启用/禁用特定功能

## 🤝 兼容性

- ✅ **服务端兼容**：可在服务器安装使用
- ✅ **多人游戏**：所有客户端均可正常使用
- ⚠️ **其他模组物品**：自动识别并分类，支持模组物品扩展
