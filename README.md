## 📖 简介

便捷合成 (Convenient Crafting) 是一个优化型 Minecraft 模组，面向 Minecraft 1.21.1 与 NeoForge 21.1.220+。它不是增加大量新物品，而是专注于优化原有的背包管理体验，通过智能整理、便携合成和配方检索等功能，让冒险、建造和探索过程更加顺手。

## ✨ 主要功能

### 🎯 一键整理背包（核心功能）

在背包或容器界面添加一个**"整理"按钮**，点击即可智能整理所有物品。

**智能整理规则：**
- ✅ **自动堆叠**：自动合并可堆叠的物品
- 📦 **智能分类**：按预设类别分类存放
  - 🔧 工具类（镐、斧、铲等）
  - ⚔️ 武器与盔甲
  - 🍖 食物类
  - 🪵 建材类（方块）
  - ⚙️ 红石类
  - 💎 战利品与贵重物品
  - 🌱 农业类（种子、农作物）
  - 📚 杂物类
- 🎒 **快捷栏优化**：总是将常用工具（镐、剑）放在快捷栏附近
- 🧱 **同类相邻**：将同类方块（石头、深板岩、安山岩等）放在相邻位置

### 🛠️ 便携式合成助手

**快捷键 `G`** 呼出简易合成面板，悬浮在游戏画面一侧。

**核心优势：**
- 🎒 **直接使用背包材料**：无需取出物品
- 🏃 **随时随地合成**：不打开工作台即可合成
- 📋 **常用配方快捷访问**：
  - 基础材料：木棍、木板、火把
  - 工具类：石制/铁制工具
  - 战斗类：箭、弓
  - 建筑类：梯子、栅栏
  - 实用类：箱子、熔炉

**适用场景：**
- 🕳️ 洞穴探险时快速补充工具
- 🏗️ 建筑过程中即时合成材料
- ⚔️ 战斗紧急时刻快速制作武器
- 🏕️ 野外生存时便捷制作必需品

### 🔎 配方检索与筛选

便携合成界面会读取当前世界中的标准合成配方，并结合玩家背包材料显示可合成状态。

**检索能力：**
- 🔍 **名称搜索**：按物品名称快速过滤配方
- ✅ **可合成筛选**：只显示当前材料足够的配方
- 📊 **状态统计**：显示当前可合成数量和配方总数
- 🧩 **配方兼容**：支持数据包或其他模组注册的标准合成配方

## 🎮 使用指南

### 安装要求
- **Minecraft**: 1.21.1
- **Mod Loader**: NeoForge 21.1.x
- **Java**: Java 21+

多版本维护和移植检查清单见 [`VERSIONING.md`](VERSIONING.md)。当前发布包只声明支持已测试的 Minecraft 1.21.1。

### 安装步骤
1. 确保已安装对应版本的 NeoForge
2. 将构建出的模组 `.jar` 文件放入 `.minecraft/mods` 文件夹
3. 启动游戏，享受整理乐趣！

### 操作说明

#### 背包整理
1. 打开背包界面（默认 `E` 键）
2. 点击右下角的 **"整理"按钮**
3. 等待自动整理完成

#### 便携合成
1. 按下 **`G` 键** 打开合成面板
2. 选择需要合成的物品
3. 确认合成（自动消耗背包中的材料）

## 🔧 配置选项

模组使用 NeoForge 配置文件（`.minecraft/config/convenientcrafting-common.toml`），可根据后续版本配置项进行调整。

快捷键可在 Minecraft 的按键设置中调整：

- ⌨️ 打开便携合成界面：默认 `G`
- 🗂️ 按键分类：便捷合成

## 🤝 兼容性

- ✅ **服务端兼容**：可在服务器安装使用
- ✅ **多人游戏**：所有客户端均可正常使用
