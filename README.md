## 📖 简介

便捷合成 (Convenient Crafting) 是一个面向 Minecraft 1.21.1 与 NeoForge 21.1.220+ 的轻量生存工作流 QoL 模组。它的核心不是增加大量新物品，而是让玩家在冒险、建造、挖矿和整理材料时少打断当前节奏：随时打开合成助手，检索配方，直接使用背包、已打开容器和可选分类袋中的材料完成合成。

模组的主玩法是**随身合成助手**：配方检索、可合成判断、批量合成、嵌套合成和材料来源整合都围绕它展开。背包/容器整理和分类袋则属于材料准备层，帮助你更快把材料整理到可以使用的状态。

## ✨ 主要功能

### 🛠️ 随身合成助手（核心功能）

**快捷键 `G`** 呼出简易合成面板，随时查看配方并直接合成，不必先摆工作台或来回翻箱子。

**核心优势：**
- 🏃 **随时随地合成**：在探索、挖矿、建造途中快速处理常用合成
- 🔎 **配方检索**：按名称、物品、模组、标签、提示文本、材料和创造标签搜索配方
- ✅ **可合成判断**：自动根据当前材料显示哪些配方可以立刻制作
- 🧩 **嵌套合成**：缺少中间产物时，可尝试自动推导合成链，并显示缺少的原材料
- 📦 **材料来源整合**：可使用玩家背包、当前打开容器，以及配置允许时的分类袋内容
- ⚒️ **多工作站支持**：支持合成台、锻造台、酿造台和切石机配方

**适用场景：**
- 🕳️ 洞穴探险时快速补充火把、工具和基础材料
- 🏗️ 建筑过程中即时合成楼梯、栅栏、箱子等材料
- ⚔️ 战斗或探索中临时制作消耗品
- 📦 整理仓库时直接从箱子材料里完成合成

### 🔎 配方检索与筛选

随身合成界面会读取当前世界中的配方，并结合可用材料显示可合成状态。

**检索能力：**
- 🔍 **名称搜索**：按物品名称快速过滤配方
- 🧭 **高级过滤**：支持按模组、标签、提示文本、创造标签和材料搜索
- ✅ **可合成筛选**：只显示当前材料足够的配方
- 📊 **状态统计**：显示当前可合成数量和配方总数
- 🧩 **配方兼容**：支持数据包或其他模组注册的兼容配方类型

### 🧱 轻量整理与材料准备

整理功能用于减少材料准备阶段的重复操作，不是模组的主玩法。

在背包或支持的容器界面添加一个**"整理"按钮**，点击即可整理物品；也可以使用鼠标中键快捷整理。

**智能整理规则：**
- ✅ **自动堆叠**：自动合并可堆叠的物品
- 📦 **智能分类**：按预设类别分类存放
  - 🔧 工具类（镐、斧、铲等）
  - ⚔️ 武器与盔甲
  - 🍖 食物类
  - 🪵 建材类（方块）
  - ⚙️ 红石类
  - 💎 战利品与贵重物品
  - 🌱 农业类（种子、农作物）
  - 📚 杂物类
- 🎒 **快捷栏优化**：总是将常用工具（镐、剑）放在快捷栏附近
- 🧱 **同类相邻**：将同类方块（石头、深板岩、安山岩等）放在相邻位置

### 🎒 分类袋（材料准备层）

模组包含种子袋、染料袋和矿物袋，用于在生存流程中减少材料散落。分类袋可以通过配置决定是否显示、是否可合成、是否自动拾取，以及袋内材料是否参与随身合成。

## 🎮 使用指南

### 安装要求
- **Minecraft**: 1.21.1
- **Mod Loader**: NeoForge 21.1.x
- **Java**: Java 21+

多版本维护和移植检查清单见 [`VERSIONING.md`](VERSIONING.md)。当前发布包只声明支持已测试的 Minecraft 1.21.1。

### 安装步骤
1. 确保已安装对应版本的 NeoForge
2. 将构建出的模组 `.jar` 文件放入 `.minecraft/mods` 文件夹
3. 启动游戏，按 `G` 打开随身合成助手

### 操作说明

#### 随身合成
1. 按下 **`G` 键** 打开合成面板
2. 搜索或选择需要合成的物品
3. 点击合成按钮，自动消耗可用材料
4. 缺少中间产物时，可尝试嵌套合成

#### 背包/容器整理
1. 打开背包或支持的容器界面
2. 点击 **"整理"按钮**，或使用鼠标中键
3. 如配置允许，按住 Alt 整理背包时可压缩矿物材料

## 🔧 配置选项

模组使用 NeoForge 配置文件（`.minecraft/config/convenientcrafting-common.toml`），可以按整合包需求调整合成助手、整理和分类袋行为。

常用配置包括：

- `enabledBuiltinRecipeTypes`：控制内置工作站配方类型
- `additionalRecipeTypes`：添加兼容的自定义配方类型
- `recipeTypeUnlockItems`：配置配方类型解锁物品
- `enableInventorySorting`：是否启用玩家背包整理
- `enableContainerSorting`：是否启用容器整理
- `enableCategoryBags`：是否显示并允许合成分类袋
- `enableBagAutoPickup`：是否自动拾取进分类袋
- `useBagContentsForCrafting`：袋内材料是否参与随身合成
- `enableAltMaterialCompaction`：Alt 整理时是否压缩矿物材料

快捷键可在 Minecraft 的按键设置中调整：

- ⌨️ 打开便携合成界面：默认 `G`
- 🗂️ 按键分类：便捷合成
