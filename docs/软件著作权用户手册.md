# 便捷合成 (Convenient Crafting) 用户手册

## 1. 软件概述

便捷合成 (Convenient Crafting) 是一款运行于 Minecraft 1.21.1 与 NeoForge 21.1.220+ 环境的游戏辅助模组软件。软件围绕玩家背包管理和常用合成流程提供辅助功能，包括背包物品整理、合成配方浏览、可合成物品筛选、背包材料直接合成等。

本软件的主要目标是减少玩家在生存、探索和建造过程中的重复背包操作，提高物品归类、查找和合成效率。

## 2. 软件信息

- 软件名称：便捷合成 (Convenient Crafting)
- 软件简称：便捷合成
- 软件版本：V1.0.0
- 模组 ID：convenientcrafting
- 运行平台：Minecraft Java Edition 1.21.1
- 依赖环境：NeoForge 21.1.220+、Java 21+
- 软件类型：Minecraft 模组软件

## 3. 安装环境

用户需要预先安装以下环境：

1. Minecraft Java Edition 1.21.1。
2. 与 Minecraft 1.21.1 匹配的 NeoForge 21.1.220 或更高兼容版本。
3. Java 21 或更高兼容运行环境。

## 4. 安装步骤

1. 启动一次已安装 NeoForge 的 Minecraft 客户端，使游戏目录生成 `mods` 文件夹。
2. 将便捷合成构建生成的 `.jar` 文件复制到 `.minecraft/mods` 目录。
3. 启动 Minecraft，并在模组列表中确认出现“便捷合成 (Convenient Crafting)”。
4. 进入游戏世界后即可使用背包整理和便携合成功能。

## 5. 主要功能

### 5.1 背包整理

用户打开背包或容器界面后，可以点击界面中的“整理背包”按钮。软件会根据物品堆叠状态和预设分类规则，对背包内物品进行整理。

背包整理功能包括：

- 合并可堆叠物品。
- 按工具、武器、食物、方块、材料等类别进行排序。
- 将相同或相近类别的物品排列到相邻位置。
- 优先保留常用物品的易访问位置。

### 5.2 便携合成

用户可以按默认快捷键 `G` 打开便携合成界面。该界面会读取当前世界中已注册的合成配方，并结合玩家背包材料显示可合成状态。

便携合成功能包括：

- 显示可用合成配方列表。
- 搜索配方名称或物品名称。
- 筛选当前背包材料可合成的配方。
- 直接消耗背包内材料完成合成。
- 支持通过数据包或其他模组注册的标准合成配方。

### 5.3 配方状态提示

软件会统计当前配方列表中可合成的数量，并在界面上显示可合成比例。用户可以通过该信息快速判断当前背包材料能够制作哪些物品。

### 5.4 快捷键设置

便携合成界面默认通过 `G` 键打开。用户可以在 Minecraft 的按键设置中找到“便捷合成”分类，并根据个人习惯修改快捷键。

## 6. 基本操作流程

### 6.1 使用背包整理

1. 进入游戏世界。
2. 打开玩家背包或容器界面。
3. 点击“整理背包”按钮。
4. 等待软件完成物品分类和排序。

### 6.2 使用便携合成

1. 进入游戏世界。
2. 按下快捷键 `G` 打开便捷合成界面。
3. 在配方列表中选择目标物品。
4. 根据可合成状态判断材料是否足够。
5. 点击合成按钮完成制作。

### 6.3 搜索和筛选配方

1. 打开便携合成界面。
2. 在搜索框中输入物品名称关键字。
3. 勾选或取消“可合成”筛选项。
4. 浏览筛选后的配方列表。

## 7. 配置说明

软件配置文件位于 Minecraft 配置目录中，文件名为 `convenientcrafting-common.toml`。用户可以根据需要调整配置项。修改配置后，部分设置可能需要重新进入世界或重启游戏后生效。

## 8. 常见问题

### 8.1 模组未显示

请确认 Minecraft 版本、NeoForge 版本和模组文件版本匹配，并确认 `.jar` 文件已放入正确的 `mods` 目录。

### 8.2 快捷键无法打开界面

请进入 Minecraft 按键设置，检查“便捷合成”分类下的快捷键是否与其他功能冲突。如有冲突，可将其改为其他按键。

### 8.3 配方显示为空

请确认已经进入游戏世界。配方数据依赖当前世界的配方管理器，未进入世界时可能无法读取完整配方列表。
